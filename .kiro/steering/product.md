# Product Overview

RSS-Tok is a modern RSS feed reader that resurrects the "dead" technology of RSS feeds by transforming them into a TikTok-style vertical scrolling experience.

## Core Value Proposition

- **No algorithms**: Users control their feed, not an AI
- **Privacy-first**: No tracking or data collection
- **Open web**: Works with any RSS feed
- **Modern UX**: TikTok-style interface meets open web values

## Key Features

- Vertical scrolling card-based feed with snap navigation
- Multi-feed mixing (combine multiple RSS sources into one randomized stream)
- Like, save, and bookmark articles
- Read tracking with hide/show read articles
- Statistics dashboard for reading habits
- Full article reading inline
- Dark theme with glassmorphism effects
- Keyboard navigation support

## Target Audience

Users who value open web principles but want a modern, engaging reading experience similar to social media apps.
